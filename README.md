# sudo-rm--rf

## AI 기반 지역사회 감염병 확산 예측 및 예방 플랫폼 프로젝트 계획서

### 프로젝트 개요

### 프로젝트명
**EpiGuard (Epidemic Guardian)** - AI 기반 지역사회 감염병 확산 예측 및 예방 플랫폼

### 목표
- 실시간 감염병 확산 위험도 예측
- 개인 맞춤형 예방 정보 제공
- 지역사회 감염병 초기 대응 지원
- SDG 3 (건강과 웰빙) 달성 기여

## 기술 스택

### 프론트엔드
- **웹**: React.js (TypeScript)
  - UI 라이브러리: Material-UI / Ant Design
  - 상태관리: Redux Toolkit
  - 차트: Chart.js / D3.js
  - 지도: Google Maps API / Mapbox
- **모바일**: Flutter
  - 상태관리: Provider / Riverpod
  - 로컬 저장소: Hive / SharedPreferences
  - 푸시 알림: Firebase Cloud Messaging

### 백엔드
- **API 서버**: FastAPI (Python)
  - 데이터베이스 ORM: SQLAlchemy
  - API 문서화: 자동 Swagger 생성

### 데이터베이스
- **주 데이터베이스**: MySQL

### AI/ML
- **허깅페이스 모델**: 기존 훈련된 모델 활용
- **라이브러리**: scikit-learn, pandas, numpy, transformers
- **간단한 예측 모델**: Linear Regression, Random Forest
- **자연어 처리**: Hugging Face Transformers

### 인프라
- **로컬 개발**: Docker (선택사항)
- **간단한 배포**: 단일 서버 또는 로컬 실행

## 시스템 아키텍처

### 전체 구조 (간소화)
```
[클라이언트] → [FastAPI 서버] → [허깅페이스 모델] → [MySQL]
     ↓              ↓                ↓
[React Web]    [간단한 AI 로직]   [로컬 데이터]

```

- **API 서버**: FastAPI (단일 서버)
- **AI 처리**: 허깅페이스 모델 + 간단한 ML
- **데이터 저장**: MySQL

## 주요 기능

### 핵심 기능
1. **실시간 확산 위험도 대시보드**
   - 지역별 위험도 맵 시각화
   - 확산 예측 그래프
   - 실시간 통계 정보

2. **개인 맞춤형 예방 가이드**
   - 위치 기반 위험도 알림
   - 개인 건강 상태 고려한 맞춤 조언
   - 행동 수칙 및 예방법 제공

3. **의료진 지원 기능**
   - 선별진료소 위치 정보
   - 백신 접종 센터 안내
   - 의료진 전용 상세 분석 데이터

4. **조기 경보 시스템**
   - 소셜 미디어 모니터링
   - 이상 징후 자동 감지
   - 관련 기관 자동 알림

### AI 모델별 세부 기능

#### 확산 예측 AI (간소화)
- **허깅페이스 모델**: 기존 시계열 예측 모델 활용
- **간단한 ML**: Random Forest + 선형 회귀
- **입력 데이터**: 가상/샘플 데이터로 시연
- **출력**: 7일 확산 예측 (시각화)

#### 맞춤형 정보 제공 AI
- **허깅페이스**: 텍스트 분류 모델 활용
- **간단한 룰 기반**: 나이, 지역별 기본 추천
- **출력**: 미리 정의된 예방 수칙 매칭

#### 이상 징후 감지 AI
- **허깅페이스**: 감정 분석 / 텍스트 분류 모델
- **샘플 데이터**: 가상의 소셜미디어 텍스트
- **출력**: 이상 징후 점수 (데모용)

## 데이터 관리 (대회용 간소화)

### 데이터 소스
- **샘플 데이터**: CSV 파일로 준비된 가상 데이터
- **공공 데이터**: 무료로 접근 가능한 일부 공개 데이터

4. **모델 입력**: 허깅페이스 모델에 바로 입력
